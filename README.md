# Приложение чата | Yandex Practicum
Приложение чата написано с использованием шаблонизатора [Handlebars](https://handlebarsjs.com/) и препроцессора [SCSS](https://sass-lang.com/).

## Запуск приложения
Установка зависимостей
```bash
npm i
```

Запуск проекта
```
npm run start
```

Сборка проекта
```
npm run build
```
